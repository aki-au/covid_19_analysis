Analysis of Trends during COVID-19
This website contains the interactive graphs created to analyze COVID-19 Trends

COVID Stringency Index

This graph analyzes how different countries' COVID Stringency Index has changed as time progresses. To see individual countries' trends, please use the drop-down menu to do so. A higher score indicates a stricter response.

COVID Face Covering Policy

This graph analyzes how different countries' face covering policies have changed as time progresses. To see individual countries' trends, please use the drop-down menu to do so. Here, 0 means no policy, 1 means that it is recommended, 2 means that it is required in some public spaces, 3 means that it is required in all public spaces, and 4 means that it is required outside the home at all times.

COVID Containment Index

This graph analyzes how different countries' containment index has changed as time progresses. To see individual countries' trends, please use the drop-down menu to do so. This index build on the stringency index, but it also includes other factors such as vaccine policy, face-covering policy, and contact-tracing.

Number of New Cases per Million

This graph analyzes how the number of new cases per million has changed as time progresses. To see individual countries' trends, please use the drop-down menu to do so.

Number of New Deaths per Million

This graph analyzes how the number of new deaths per million has changed as time progresses. To see individual countries' trends, please use the drop-down menu to do so.

School Closure Policy

This graph analyzes how different countries' school closure policy has changed as time progresses. To see individual countries' trends, please use the drop-down menu to do so. Here, 0 means no policy, 1 means that school closure is recommended, 2 means that it is required for some levels, 3 means that it is required for all levels.

Vaccine Availability

This graph analyzes how different countries' school closure policy has changed as time progresses. To see individual countries' trends, please use the drop-down menu to do so. There are 3 groups to keep in mind here: key workers, clinically vulnerable groups and the elderly. Here, 0 means no vaccine availability, 1 means that there is availability for one of these groups, 2 means that there is availability for two of these groups, 3 means that there is availability for three of these groups, 4 means that there is an availabilty of the vaccine for additional select groups as well, and 5 means that there is universal availability of vaccine
